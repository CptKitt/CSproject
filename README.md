 **Final Submission**
 
 Includes the JavaFX version as GUI/GUIMain.java, and the text-based version as Text/TextMain.java.
 To run either:
 - Navigate to /src
 
 - run "javac GUI/GUIMain.java"
 - run "java GUI.GUIMain"
 
 or
 
 - run "javac Text/TextMain.java"
 - run "java Text.TextMain"
